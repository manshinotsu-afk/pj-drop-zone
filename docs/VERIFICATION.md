## 4. 手動で確認してほしい項目

1. **起動**  
   - `index.html` をブラウザで開く、または `npx serve .` で表示した URL を開く。
2. **タイトル画面**  
   - 「DROP ZONE」「落とすか、落とされるか」と 1P アイコン（playerFace.png）が表示されること。
3. **スタート**  
   - Space または「Tap to Start」でゲーム開始。READY → FIGHT!! の後に盤面が表示されること。
4. **1P 表示**  
   - キャンバス上で 1P がスプライト（向き・idle/移動）で表示されること。
5. **操作**  
   - 矢印キー / WASD で移動、Space で攻撃。攻撃ゲージ・CPU の動きが問題ないこと。
6. **リザルト**  
   - 勝敗後にリザルト画面（playerFace アイコン・勝利/敗北/引き分け表示）が出ること。Space で再戦できること。
7. **スマホ表示（任意）**  
   - 仮想 D-pad と Attack ボタンが表示され、タップで操作できること。

# DROP ZONE

10×10 のグリッドで繰り広げる 4 人対戦ゲーム。プレイヤー（1P）が 3 体の CPU と戦い、攻撃でパネルを落として相手を落とす。最後までパネルに残った 1 体が勝ち。

**落とすか、落とされるか。**

---

## 遊び方

こちらから遊べる。
https://manshinotsu-afk.github.io/pj-drop-zone/

- **移動** … グリッド上を移動。パネルの上に立っているとき、そのパネルが落ちると自分も落下してゲームオーバー。
- **攻撃** … 向いている方向に攻撃し、一直線のパネルを落とす（線上にいる敵もまとめて落下）。
- 相手を落とし、自分だけ残れば勝利。

---

## 操作

| 操作       | PC / デスクトップ     | スマホ           |
|------------|------------------------|------------------|
| 移動       | W A S D / 矢印キー     | 画面の D-pad     |
| 攻撃       | Space                  | 攻撃ボタン       |
| スタート / 再戦 | Space            | タップ           |

- **タイトル画面** … Space またはタップでスタート。
- **リザルト画面** … Space またはタップで再戦。
